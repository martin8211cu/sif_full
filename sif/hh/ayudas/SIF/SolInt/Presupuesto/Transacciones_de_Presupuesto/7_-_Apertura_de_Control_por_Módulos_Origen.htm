Apertura de Control por Módulos Origen

En esta opción se pueden asociar módulos para permitir el control abierto para la reserva de presupuesto. El control abierto no controla el presupuesto, esto quiere decir que puede haber disponibles negativos y que permite la aprobación de excesos. Un ejemplo de un módulo que se le abra el control puede ser el de pago de nómina.
