Formular Versión Usuario

En esta opción se listarán todas aquellas versiones creadas en la opción Versiones de Formulación de Presupuesto, que tengan asociada la etapa Usuario.

 

Una vez que se tiene la base total del periodo a presupuestar, se le entrega al usuario para que solicite lo que crea necesario cambiar con respecto a la base, el usuario puede digitar el ajuste que desea.

 

 

Al seleccionar una formulación de presupuesto, se desplegará la siguiente pantalla donde se podrá modificar el monto solicitado realizándole ajustes de usuario (aumentándole o rebajándole el monto):

 

 

Para ello se debe de seleccionar una cuenta de mayor y a continuación se desplegará la lista de cuentas de presupuesto para proceder a cambiar los montos solicitados, realizándole ajustes de usuario (aumentando o rebajando el monto):

 

 

Una vez en esta pantalla, se selecciona la cuenta a modificar y se presenta la pantalla donde se puede escoger, si hacer el ajuste por moneda o por meses:

 

 

La columna que se cambiará será la de Ajustes Usuario, ya sea aumentando o rebajando el monto solicitado:
