Reasignar Cargas de Trabajo

(Solicitudes de Compra)

Este proceso reasigna solicitudes de compra de un comprador a otro.

 

Lo único que se debe de hacer es seleccionar el comprador para que se le despliegue la lista de solicitudes que tiene asignadas hasta el momento. Luego seleccionar de la lista la solicitud que se quiere reasignar, para que se despliegue la lista de compradores y seleccionar al que se le va a asignar dicha solicitud.
