The soEditor Family

 

soEditor is offered in five editions:

 

Lite:

 

  • soEditor Lite 2.5

    soEditor Lite 2.5 is a scaled down version of soEditor Pro.  It is intended to give users a chance to try soEditor at no cost.  It is licensed for a single development and a single production server installation and allows for unlimited users and unlimited URLs.  The ability to modify the source code is not included.

  • soEditor Lite 2.5 Source

    soEditor Lite 2.5 Source is a scaled down version of soEditor Pro.  It is licensed for a single development and a single production server installation and allows for unlimited users and unlimited URLs.  The ability to modify the source code is also included.

 

Professional:

 

  • soEditor Pro 2.5

    soEditor Pro 2.5 is the full featured version of soEditor. It is licensed for a single development and a single production server installation and allows for unlimited users and unlimited URLs.  The ability to modify the source code is not included.

  • soEditor Pro 2.5 Source

    soEditor Pro 2.5 Source is the full featured version of soEditor.  It is licensed for a single development and a single production server installation and allows for unlimited users and unlimited URLs.  The ability to modify the source code is also included.

 

Enterprise:

 

  • soEditor Pro 2.5 Enterprise

    soEditor Pro 2.5 Enterprise allows for an unlimited server installation within the same physical network, unlimited users and unlimited URLs.  The ability to modify the source code is also included.  This license is designed for organizations wanting to have the freedom to deploy soEditor Pro on as many servers as needed without having to purchase additional individual licenses.
