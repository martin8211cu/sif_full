Uso del Aplicativo

Lista de selección

Una lista de selección es una relación con los valores disponibles para un catálogo de referencia, este catálogo usualmente tiene los valores posibles y vigentes del catálogo externo, bastando elegir el indicado para la establecer la referencia.  

Por ejemplo si se requiere guardar una referencia a un país, se debe seleccionar este de una lista de valores posibles, cargados, en este caso, en los parámetros de operación del sistema.

Selección

El uso de una lista de selección se indica con el icono siguiente, desplegado a la derecha del campo de referencia, para su uso basta seleccionar el icono y una lista con los valores válidos será desplegada, una vez encontrado el valor deseado en esta lista, bastará marcar el mismo, la lista se cerrará y el valor se relacionará con el campo de referencia.

Los usuarios expertos pueden alternativamente ingresar directamente el valor en algunos casos, pero sin importar el método elegido se validará el valor ingresado en el campo.

Validación

El valor del campo de referencia se valida contra los valores disponibles en el catálogo referido, indicando un error cuando se ingrese un valor fuera de este ámbito.

Nunca se podrá guardar el registro con un valor no validado para un campo.
