Grupo de Cobro

Concepto

Son una Agrupación de las cuentas en diferentes ciclos de cobro de la empresa.  Cada empresa define diferentes ciclos de cobro de acuerdo a sus intereses.

Los cobros a las cuentas son un aspecto administrativo, sin embargo su clasificación se define a nivel operativo para facilitar los cambios a la misma como servicios del proceso atencional.

Administración

Los Grupos de Cobro se administran mediante una lista, ordenada por el código asignado a cada Grupo de Cobro, en esta se desplegarán los siguientes campos o columnas:

  • Código de Grupo de Cobro Es el valor que identifica de forma única cada grupo de cobro. Es determinado por la empresa para efectos de integración con otros sistemas y procesos de gestión administrativa.

  • Nombre de Grupo de Cobro Nombre de cada Grupo de Cobro codificado, para su uso por parte del usuario del sistema. . El valor de este campo describe cada Grupo de Cobro sin permitir valores nulos.

 

Acciones sobre Grupos de Cobro

Acción

Cómo se realiza

Consecuencia

Agregar

Agrega un nuevo Grupo de cobro para su selección en las cuentas que lo relacionen, invocando un formulario de registro.

Cambiar

Modifica o inhabilita un Grupo de Cobro existente, precargando un formulario de registro con los valores vigentes del mismo.

 

Registro

El registro de un nuevo Grupo de Cobro o el mantenimiento de uno existente, se realiza mediante un formulario de registro, donde se captura la siguiente información.

  • Código de Grupo de Cobro Es el valor que identifica de forma única cada grupo de cobro. Es determinado por la empresa para efectos de integración con otros sistemas y procesos de gestión administrativa.

  • Nombre de Grupo de Cobro Nombre de cada Grupo de Cobro codificado, para su uso por parte del usuario del sistema. . El valor de este campo describe cada Grupo de Cobro sin permitir valores nulos.
