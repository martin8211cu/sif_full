Requirements

Internet Explorer 5.0 or above with JavaScript enabled and a Windows operating system is required to use soEditor's WYSIWYG editing functionality.  If the client browser or operating system is not supported a simple text area will be displayed instead.

 

User System Requirements

 

  • Windows 95/98/ME/NT/2000/XP

  • Internet Explorer 5.0 or above

  • JavaScript must be enabled

 

Server System Requirements

 

  • ColdFusion Server 4.5+
