What's new in version 2.5

We've added several new features and improvements to this release including the following:

 

  • Multiple instances of soEditor can now be used in a single form
     

  • A new anchor tool that lets you create and manage anchor links in your content
     

  • Right-click context menu support for the most common editing tasks.
     

  • The initial focus can now be set to another field in your form
     

  • New example applications and help documentation
     

  • Many bug fixes and performance enhancements
